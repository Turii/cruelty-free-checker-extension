# Tech Stack

## Frontend (Extension UI)
- HTML (popup UI)
- CSS (basic styling)
- JavaScript (logic)

## Chrome Extension
- Manifest v3
- Content Script (to read page content)
- Popup script (user interaction)

## Data Storage
- Local JSON file (brands list)
- Stored inside extension (no backend)

## AI (optional, later)
- Use AI to analyze unknown brands (future improvement)

## Why this stack?
- Simple and fast to build
- No backend needed
- Lightweight for users
