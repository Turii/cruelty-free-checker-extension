# Cruelty-Free Checker Chrome Extension

A simple Chrome extension that detects a brand on product pages and shows whether it is cruelty-free.

## How to use

1. Download or clone the repository
2. Open Chrome → chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the `extension` folder

## Disclaimer
This is a simple MVP project using a small local dataset.
